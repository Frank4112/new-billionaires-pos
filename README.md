# NEW BILLIONAIRES POS SYSTEM

A modern full-stack Point of Sale (POS) and Inventory Management System for bars, restaurants, and retail businesses.

## Features
- Product management
- Inventory tracking
- POS sales system
- Reports and analytics
- User authentication
- Responsive dashboard

## Tech Stack
### Frontend
- React
- Vite
- React Router DOM

### Backend
- Node.js
- Express.js

### Database
- MySQL

## Current Progress
Frontend architecture, product management, and the first backend API are currently under development.

## Backend API
Install backend dependencies:

```bash
cd backend
npm install
```

Run the backend API:

```bash
npm start
```

Product endpoints:

- `GET /api/products`
- `GET /api/products/:id`
- `POST /api/products`
- `PUT /api/products/:id`
- `DELETE /api/products/:id`
